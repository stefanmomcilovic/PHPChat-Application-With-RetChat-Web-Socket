The main files for the messages server are in the src folder, in order to start the message server in the console you have to type "php src/server.php" and in the console it will say if the connection is successful where is installed server on what file you put it, everything is set in Chat.php file in order to save messages in the mysql database, everything else is done via javascript and php, look at the files chatroom.php and private.php
Everything is done object oriented over the classes, to check there is no update write composer update in the console, you need to have vendor file and composer files in folder anything else you can set up by following this example.
